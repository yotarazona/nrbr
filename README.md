
# NRBR GitHub Page

This site presents the Normalized Radar Burn Ratio (NRBR), a radar-based index for burned area detection using Sentinel-1 SAR data.

## 🚀 Deployment Instructions
1. Create a GitHub repository named `NRBR`
2. Upload all files in this ZIP (including folders)
3. Go to repository settings > Pages
4. Choose branch: `main` and folder `/ (root)`
5. GitHub will deploy your site at:
   https://<your-username>.github.io/NRBR/
